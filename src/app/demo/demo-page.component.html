SystemExperiments — CmdSectionComponent

Sticky CMD header used at the top of each board. Two multi-select dropdowns (Side, Wheel) emitting a fresh CmdSelection whenever either changes. Stateless — the shell owns the selection.

SystemExperiments — BoardFooterComponent

Singleton action bar with Defaults / Cancel / Apply. The shell mounts ONE instance outside the tab-group and routes the three events to the active board (test ids: footer-defaults, footer-cancel, footer-apply) — there is no per-board footer.
