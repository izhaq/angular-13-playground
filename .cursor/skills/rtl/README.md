# rtl-bidi-ui

Portable skill package for generating and reviewing direction-safe HTML and CSS for both RTL and LTR interfaces.

Contents:
- `SKILL.md` — primary instructions and invocation metadata
- `references/rtl-checklist.md` — acceptance checklist
- `references/css-logical-properties.md` — physical-to-logical migration map
- `references/html-bidi-patterns.md` — markup patterns for mixed-direction content
- `references/review-template.md` — preferred output structure
