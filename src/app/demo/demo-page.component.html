EngineSim — CmdSectionComponent

Sticky CMD header used at the top of each board. Two multi-select dropdowns (Side, Wheel) emitting a fresh CmdSelection whenever either changes. Stateless — the shell owns the selection. 

selection = {{ cmdSelection | json }}

EngineSim — BoardFooterComponent

Sticky board footer with Defaults / Cancel / Apply. Each button stamps a namespaced data-test-id via the boardId input so primary and secondary footers stay unique even when both tabs are mounted at the same time. 

recent events:
{{ footerEvents.length ? (footerEvents | json) : '(none yet — click a button above)' }}

EngineSim — StatusGridComponent (8 cols)

Primary board's grid: row labels in the leading column, then 8 data columns (L1-L4 + R1-R4). Hover a cell for column highlight; click a cell to select it. Inspect any cell to see grid-primary-{{ '{' }}fieldKey{{ '}' }}-{{ '{' }}colId{{ '}' }}.

EngineSim — StatusGridComponent (11 cols)

Secondary board's grid: same component, just 11 columns this time (L1-L4 + R1-R4 + TLL + TLR + GDL). Sparse rows are fine — cells with no value render empty.
